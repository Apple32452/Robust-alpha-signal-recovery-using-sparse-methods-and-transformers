MATLAB project: Robust Alpha Signal Recovery Using Transform-Domain Sparsity

Quick start:
1) Put CSV price files in a folder named "data" next to run_alpha_project.m
2) Required columns in each CSV: Date and Adj Close (or Close)
3) In MATLAB:
   cd('path/to/robust_alpha_matlab')
   run_alpha_project

If no data folder exists, the script can create synthetic price data automatically.
